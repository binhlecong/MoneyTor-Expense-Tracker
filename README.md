# Moneytor
An Expense Tracker Application for Android Mobile

# Features
* Manage spending
* Manage debt/lend
* Manage spend goal
* Analyze spending (summary by time period, categories)
* Visualize by chart
* Backup, sync data
* Support Vietnamese, English, Japanese language
* Support dark and light mode
